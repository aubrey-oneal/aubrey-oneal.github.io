Favorite tools

GitHub Projects

I believe that multi-disciplinary teams can learn a lot from developer workflows. This is why I favor using Agile ceremonies, triaging work in GitHub projects, and collaborating where the code lives.

Figma

I use and maintain Figma design kits and documentation every day. I believe in prototyping with stakeholders directly in Figma over creating slide decks.

Schemas

Challenges in enterprise design systems often stem from lack of alignment across properties. I enjoy solving such problems with the use of a schema, which standardizes definitions and documentation in a structured format that can be used across websites and other services.

User Research

It's important to make decisions based on user testing and metrics. I enjoy conducting both quantitative and qualitative research, and often find creative ways to support a user research program on teams that are moving fast.
